# Shared Assets

## Styles
- `assets/css/curriculum.css`: Base theme, badges, progress bar, sections, code example blocks.

## Scripts
- `assets/js/navigation.js`: Section navigation activation and smooth scrolling.
- `assets/js/progress.js`: Reading progress indicator.
- `assets/js/code-runner.js`: Copy code and guide to run in notebooks.

## Usage
- Link CSS and JS from lessons to unify design and behavior.
- Keep page weight under 1MB and mobile-first.
